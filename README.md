# Reto SSTI Básico – Formación Presencial ECSC (España)

Este repositorio contiene un reto **básico de Server-Side Template Injection (SSTI)** desarrollado para la **formación presencial de la Selección Española ECSC**.

El ejercicio fue utilizado durante una sesión organizada por **INCIBE** y **Telefónica** como parte del proceso formativo.
